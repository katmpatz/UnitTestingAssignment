# UnitTestingAssignment

Πως να εκτελέσετε τα JUnit tests:

1.Κατεβάστε το eclipse.(Αν το έχετε ήδη μπορείτε να παραλείψετε αυτό το βήμα.)

2.Καντε clone το repository στον υπολογιστή σας, δίνοντας την εντολή στο cmd: git clone https://github.com/katmpatz/UnitTestingAssignment.git

3.Για τον σκοπό της άσκησης δημιουργήστε ένα αρχείο με όνομα grades.txt, το οποίο θα περιέχει βαθμούς φοιτητών.

4.Ανοίξτε με το eclipse τον φάκελο unit_testing που βρίσκεται μέσα στον φάκελο UnitTestingAssignment.

5.Τα tests βρίσκoνται στον φάκελο src\test\java\unit_testing.

6.Για να εκτελέσετε ένα junit test κάντε δεξί κλικ πάνω του και πατήστε Run as -> JUnit Test

7.Εάν εμφανιστεί μία πράσινη μπάρα στα αριστερά της οθόνης σημαίνει ότι τα test πέτυχαν, ενώ αν η μπάρα είναι κόκκινη σημαίνει ότι απέτυχαν.

Καλή επιτυχία!
